Ultrasonic Stepper Motor Controlled Project

Team 401


Top Level Narrative




The purpose of the Ultrasonic Motor Controller design project is to control a stepper motor using an ultrasonic rangefinder. It utilizes six components, an ultrasound module, potentiometer, HCS12 chip, LCD display, stepper motor, and a push button. To read the distance of an object from the ultrasonic module, the program sends a pulse to the module, which then returns a pulse-width modulation of the measured distance.

There are five modes, POR, Forward, Reverse, Pattern, and Debug mode. POR mode means that the motor will not be moving. The pattern mode is a sequence which the motor will follow. Depending on the pattern, the motor will be switching to various o-clock positions. When we are not in distance-measuring mode, we will be in debug mode. The debug mode is similar to the operation of the distance mode, but we are now using the values of the potentiometer to control the motor.
